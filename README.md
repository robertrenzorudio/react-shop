## Run Locally

### Clone the repo
```
$ git clone https://github.com/robertrenzorudio/react-shop/
$ cd react-shop
```
---
### Install dependencies
#### With yarn
```
$ yarn or 
$ yarn install
```

#### With npm
```
$ npm install
```

---
### Run the server
#### With yarn
```
$ cd server 
$ yarn build
$ yarn start
```

#### With npm
```
$ cd server
$ npm run build
$ npm run start
```

---
### Run the React app
#### With yarn
```
$ cd web 
$ yarn start
```

#### With npm
```
$ cd web
$ npm run start
```
